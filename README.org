* Exoshell

A terminal proxy with sub-shell for programmatic controlling your via
Common-Lisp terminal. 

** Getting Started 

*** Prerequisites

This project requires ECL (Embeddable Common-Lisp) which can be found at the
link below. 

- https://common-lisp.net/project/ecl/

*** Installing 

#+BEGIN_SRC sh
autoreconf -iv
./configure 
make 
sudo make install 
#+END_SRC
